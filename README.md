## Getting started

## Run locally (bare minimum)

To serve the repository `docs/` folder locally with Python's built-in HTTP server:

```bash
cd ai-systems-notes
python serve.py
# open http://127.0.0.1:8000
```

This is intentionally dependency-free: it serves the `docs/` markdown files directly.
